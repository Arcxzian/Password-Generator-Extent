User Guide: Password Generator

  Follow these steps to generate a secure password:
  Launch the Program: Run the script by typing python Untitled-1.py in your terminal.
  Specify Length: The program will prompt you to enter a password length between 6 and 16.
  Note: The system uses Input Validation to ensure only valid numbers are accepted, acting as a safety measure for the code logic.
